The Brisbane Social

The Brisbane Social Mastodon server. Hosted in sunny Brisbane, for users from and interested in southeast Queensland.

This site is intended to be a pleasant space for folks to enjoy Mastodon. This instance is run out of Brisbane, Australia to allow faster access for locals.

Code of Conduct

For the code of conduct, see the about page on the Mastodon instance.

Banned instances

The Brisbane Social instance blocks some other Mastodon instances for reasons of threatening, graphic, sexual, or illegal content.

Please note that despite this, this is ultimately an unmoderated and global space, so if you find content that breaches the code of conduct or local/international laws, please report it and it will be dealt with swiftly.

Banned instances generally follow those of toot.cafe which follows those of mastodon.social. Other blocks will be added or removed on a case by case basis.
